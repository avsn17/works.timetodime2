## ✨ Features

### 💻 Terminal Version (`pomodoro.py`)
- **Seamless Multitasking**: Chat or check stats while the timer keeps running.
- **Auto-Save Engine**: Progress is logged every 30 seconds.
- **Rankings**: Advance from **Space Cadet** to **Galactic Overlord**.



### 🌐 Web Version (`pomodoro_web.html`)
- **Starfield Engine**: High-performance HTML5 Canvas.
- **Dual-Channel Chat**: Wisdom Bot and "Community Chat".
- **Music Autoplay**: Nahtloser Übergang in die Pause mit Lofi-Beats.
